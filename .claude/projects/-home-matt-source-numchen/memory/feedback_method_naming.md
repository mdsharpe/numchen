---
name: Method naming convention
description: User prefers methods to be named beginning with a verb
type: feedback
---

Methods should be named beginning with a verb (e.g. `GetCanMoveToDestination` not `CanMoveToDestination`).

**Why:** User's personal style preference for clarity and consistency.

**How to apply:** When naming any new public methods, ensure the name starts with a verb (Get, Set, Create, Move, Place, etc.). Properties like `IsComplete` or `IsEmpty` are fine as-is.
