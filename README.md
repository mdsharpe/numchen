## Running locally

### Prerequisites

- [.NET 10 SDK](https://dotnet.microsoft.com/download)
- [Node.js](https://nodejs.org/) (v20.19+ or v22.12+)

### VS Code (recommended)

Open the repo in VS Code and run **Terminal > Run Task... > API + UI**. This starts the API and UI in separate terminal tabs.
